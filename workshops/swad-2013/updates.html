In continuance to the commitment of ACM Student Chapter (Technical Society of Department of Information Technology) for preparing students towards the latest technologies, the Chapter plans to organize 3rd Summer Workshop on “Secure Web-based Application Development”.

3rd Summer Workshop shall familiarize students with the technologies related to web application development and securing web applications. It endeavors to cover the following “indicative” topics.

Introduction: Working of Web, HTTP, Web Browser & Server, Database Server.

HTML: Elements, Attributes, Formatting, Style, Links, Images, Tables, Lists, Forms, Frames.

CSS: Syntax, Id & Class, Background, Text, Fonts, Links, Lists, Tables.

JavaScript: Statements, Variables, Operators, If-Else, Switch, Functions, Loops.

PHP: Syntax, Variables, String, Operators, If-Else, Switch, Arrays, Loops, Functions, GET, POST.

JSP: Directives, Declarations, Pre-defined variables, Tags, Sessions.

mySQL: Syntax, Queries (Select, Update, Delete, Insert), PL-SQL.

Web security: Web attack mechanisms and counter-measures.

Getting it all Together: Example web applications

Project Discussions: Problems shall be discussed that can be solved through development of secure web application and teams shall be formed and project advisor per team shall be allocated.
